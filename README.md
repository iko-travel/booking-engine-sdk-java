# Wink Java SDK

This SDK contains 4 unique libraries you can leverage to communicate with the wink platform.

Get from Maven Central Repository.

## Affiliate
```
<dependency>
<groupId>travel.wink</groupId>
<artifactId>affiliate-sdk-java</artifactId>
<packaging>jar</packaging>
<version><!-- see Releases --></version>
</dependency>
```
